Use:Used to perform several hero commands.
Command:`{{prefix}}hero (list/suicide/show [user]/fight [user])`

Example:
`{{prefix}}hero`-Assigns a hero.
`{{prefix}}hero list`-Lists users and their heros.
`{{prefix}}hero suicide`-Kills the actual hero so that you can assign yourself another one.
`{{prefix}}hero show @Noob`-Shows Noob's hero.
`{{prefix}}hero fight @Beaver33`-You start an epic hero fight against Beaver33.

```>Don't include brackets when using the commands.```
